# Bicycle Sales & Operations Analysis

## Project Overview
This project analyzes sales transactions and operational metrics for a bicycle retailer to track revenue growth, store performance, and customer trends using interactive visualizations.

## Tools Used
- **Microsoft Power BI** (Dashboard & Visualization)
- **Power Query** (Data Cleaning & ETL)
- **DAX** (Measures & Calculations)

## Key Insights
- **Total Revenue:** Generated **$8M** across 1.6K+ orders.
- **Top Category:** Mountain Bikes are the best-selling category ($2.82M).
- **Operations:** Achieved an **89.5%** order completion rate.
- **Top Market:** **New York** is the highest performing state ($5.4M).
- **Top Brand:** **Trek** dominates sales volume compared to other brands.
